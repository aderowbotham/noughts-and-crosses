Inspired by Matthew Scroggs’ MENACE. This is my own implementation of it using the same basic principles but built from scratch as a learning exercise.

Unlike MENACE We don’t pre-fill matchboxes; rather, on encountering a board state we then look if it has been seen before otherwise we then seed the box with its starting state.
